## 問題を解く上で意識すべきこと

- 問題文を正しく理解する
- 場合分けを行う
- 計算量を見積もってから実装しはじめる
- 添字についてきちんと決めてから実装しはじめる
- 何かを固定する
  - 平面走査
- 別の問題に言い換える

### 使用例

- max は半群だから操作の取り消しができない
- 自然数は加法について閉じている
- セグ木に乗せるために、半群に単位元を定義してモノイドにする
- セグ木のモノイドの空間計算量はO(N)
